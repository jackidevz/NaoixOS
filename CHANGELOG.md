# Changelog

## NaoixSH v0.2.3-patchV2 — 2026-07-19

### Added
- New `jmdevel` command — alias for `jackmasterdev`, added for simplicity. Both commands trigger the same `run_jackmasterdev()` developer easter egg.
- New `easterg` command - alias for `hidden-easter-egg`, added either for simplicity. Both commands trigger the same output.

### Changed
- Updated `ver` command output formatting.
- Updated `about` command output formatting.
- Bumped `SHVER` from `v0.2.3-patchV1` to `v0.2.3-patchV2` in `version.h`.

### Notes
- `KEVER` (NaoixOS kernel version) remains at `v0.6.8-rc1`, unchanged in this patch.
